¿Qu√© hace la API?

La API proporciona predicciones sobre el medallero de cada país participante en los próximos Juegos Olímpicos de Tokio 2020.

Además también permite consultar gráficos y datos en bruto, entre otras funcionalidades.

Tecnologías empleadas:

  1. Gráficos: Matplotlib, Seaborn, Plotly
  2. Manipulación de datos: Pandas
  3. Contenedores: Docker y Heroku
  4. Creación de API: Flask
  5. Machine Learning: H2O, SKlearn (Google Colab)
  6. Bases de datos: MongoDB

             

Endpoints y opciones disponibles:

Endpoint Funcionalidad
/plots/<tipo>
 Genera y muestra un gráfico
/data/<tipo>
 Genera y muestra un dataframe con html y css
/predict/
 Otorga predicciones de medallero
